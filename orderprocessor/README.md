# hello-streams :: orderprocessor

## Build

```
# from top level directory
$ cd orderprocessor
$ ./mvnw clean install
```

## Run
```
# from top level directory
$ cd orderprocessor
$ java -jar target/orderprocessor-0.0.1-SNAPSHOT.jar
```

- Open a browser to http://localhost:5000/graphiql


## Test
```
# from top level directory
$ cd orderprocessor
$ ./mvnw clean test
```
